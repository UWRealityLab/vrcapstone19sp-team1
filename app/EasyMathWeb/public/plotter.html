Plotter : Type math equations below

Sphere Equation : (x - a)^2 + (y - b)^2 + (z - c)^2 = r^2

Cylinder Equation : (x - a)^2 + (y - b)^2 = r^2 where m < z < n

Cone Equation : (x - a)^2 + (y - b)^2 = r^2 / c^2 where m < z < n

Ellipsoid Equation : (x - a)^2 / u^2 + (y - b)^2 / v^2 + (z - c)^2 / w^2 = r^2
